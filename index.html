Commit Messages

It is important to save and create commit messages regularily. It is also crucial that you make them descriptive! Include things like:

  • What you've added, edited or deleted
  • The number of lines that were touched
  • What tags you've used

List Tutorial and Different Types

Unordered List

This is an unordered list. To create, type ul with < >

E.X. 1

  • 2 cups all-purpose flour
  • 2 cups sugar
  • 3/4 cup unsweetened cocoa powder
  • 2 teaspoons baking powder
  • 1 1/2 teaspoons baking soda
  • 1 teaspoon salt

Ordered List

This is an ordered list. To create, type ol with < >

inside the ul tag is the li tag. To create, type li with < > on the list item. You must use this with every list item.

Both the ol and ul tags are a parent tag, meaning additional tags will be inside it.
